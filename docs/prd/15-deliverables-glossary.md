## 20. Glosarium

| Istilah | Definisi |
|---------|----------|
| **RMI** | Remaja Masjid Istiqomah |
| **CMS** | Content Management System — panel admin untuk mengelola konten |
| **CRUD** | Create, Read, Update, Delete — operasi dasar manipulasi data |
| **JWT** | JSON Web Token — metode autentikasi stateless |
| **SSR** | Server-Side Rendering — render halaman di server |
| **SSG** | Static Site Generation — generate halaman statis saat build |
| **SEO** | Search Engine Optimization — optimasi untuk mesin pencari |
| **OG** | Open Graph — protokol metadata untuk sharing sosmed |
| **UAT** | User Acceptance Testing — pengujian penerimaan oleh stakeholder |
| **Sanlat** | Pesantren Kilat — program pesantren singkat RMI |
| **Lighthouse** | Tool Google untuk audit performa, aksesibilitas, SEO |
| **Mobile-first** | Desain dimulai dari tampilan mobile, lalu scale up ke desktop |
