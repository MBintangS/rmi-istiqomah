# Remaja Masjid Istiqomah (RMI)

Website resmi Remaja Masjid Istiqomah.

## Struktur Project

```
rmi/
├── frontend/     # Next.js 14 (App Router)
├── backend/      # Express.js API (coming soon)
└── docs/         # PRD & roadmap
```

## Frontend — Development

```bash
cd frontend
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000)
